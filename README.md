# Prerequisite's

1. Have Tk Installed on Your System

### Installing Tk Using a Package Manager

Arch:
`sudo pacman -S tk`

Ubuntu/Debian:

`sudo apt install python-tk`
 OR 
`sudo apt install python3-tk`


Fedora:
`sudo dnf install python3-tkinter`
